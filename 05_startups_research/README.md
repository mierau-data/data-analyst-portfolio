# Исследование рынка стартапов

> **Учебный проект** курса «Аналитик данных» Яндекс Практикум, 2026. Кейс инвестиционного консалтинга на обезличенных данных Crunchbase (`cb_investments`, `cb_returns`).

## Задача

Заказчик — инвестиционный консалтинг, который готовит рекомендации для портфельных инвесторов. На календаре условный 2015 год. Нужно:

- проанализировать структуру финансирования стартапов;
- понять, какие сегменты рынка стабильно растут;
- сравнить типы финансирования по объёму, популярности и доходности возвратов;
- сформулировать рекомендацию: в какую отрасль и через какой инструмент инвестировать.

## Данные

Два датасета:
- `cb_investments` — компании, рынок, сумма привлечённых инвестиций, статус, страна, даты раундов финансирования и разбивка по 13 типам инструментов (`seed`, `venture`, `angel`, `debt_financing`, `private_equity`, `post_ipo_equity` и др.).
- `cb_returns` — возвраты средств по типам финансирования и годам.

## Стек

Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook.

## Что сделано

**Предобработка:**
- приведение названий столбцов к snake_case;
- очистка `funding_total_usd` от форматирования разрядов и приведение к числовому типу;
- обработка дат, заполнение пропусков `mid_funding_at` через середину интервала между первым и последним раундом;
- удаление дубликатов и строк без данных о финансировании; оценка процента отброшенных записей.

**Инжиниринг признаков:**
- три группы по срокам финансирования (один раунд / до года / больше года), визуализация по количеству компаний и по объёму инвестиций;
- сегментация рынков на массовые (>120 компаний), средние (35–120) и нишевые (<35); средние и нишевые объединены в общие группы `mid` и `niche`.

**Работа с выбросами:**
- метод IQR в разрезе каждого сегмента, выявление компаний с аномальным финансированием;
- топ сегментов с наибольшей долей аномалий;
- проверка полноты данных за 2014 год и фильтрация: оставлены только годы, в которых зафиксировано ≥ 50 раундов финансирования.

**Анализ типов финансирования:**
- сравнение объёмов и популярности по 13 инструментам;
- сопоставление с возвратами по типам;
- расчёт нормированной доли возвратов по годам для `venture`, `debt_financing`, `private_equity`, `seed`, `angel`.

**Динамика:**
- средний размер одного раунда по годам;
- активность рынка по числу раундов;
- сводная таблица по сегментам, отбор тех, что показали рост в 2014 году к 2013;
- линейные графики динамики финансирования по растущим массовым сегментам.

## Ключевые выводы

**Сегменты с уверенным ростом в 2014 году** — Technology, Internet, E-commerce. Лидер по объёмам и темпам — **Technology**.

**Динамика рынка:** рынок рос до 2013 года, в 2014 — снижение активности и среднего чека одного раунда. Снижение среднего чека отражает структурный сдвиг: больше небольших сделок, меньше крупных.

**Возвраты по типам финансирования:**
- `private_equity` — самый устойчивый рост доли возвратов, минимальная волатильность.
- `venture` — рост есть, но с периодическими просадками.
- `seed`, `angel`, `debt_financing` — высокая волатильность, устойчивого тренда нет.

**Объём vs популярность.** Часто используемые, но скромные по объёмам: `seed`, `angel`, `equity_crowdfunding`. Редкие, но крупные по суммам: `post_ipo_equity`, `private_equity`, `debt_financing`. Это типичная картина: ранние стадии — массовый, но мелкий чек; поздние — точечные, но большие сделки.

## Рекомендация заказчику (на 2015 год)

**Отрасль для инвестирования — Technology** (с альтернативами Internet и E-commerce). Сегмент даёт максимальные объёмы, активный рынок и стабильный интерес инвесторов.

**Инструмент — private_equity** в приоритете, `venture` как альтернатива. Логика: `private_equity` показывает самую предсказуемую динамику возвратов и подходит для зрелых проектов; `venture` балансирует риск и доходность для растущих компаний. `seed`, `angel` и `debt_financing` для портфельной стратегии менее уместны — высокая волатильность возвратов.

**Ограничения выводов:** возможна неполнота данных по возвратам (задержки фиксации), высокая волатильность отдельных сегментов и неравномерная активность по годам.
