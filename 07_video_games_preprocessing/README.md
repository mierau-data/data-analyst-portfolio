# Рынок видеоигр (2000–2013) — предобработка данных

> **Учебный проект** курса «Аналитик данных» Яндекс Практикум, 2026. Кейс с условным заказчиком — командой игры «Секреты Темнолесья» — на исторических данных о продажах видеоигр.

## Задача

Команда «Секретов Темнолесья» готовит статью для IT-ресурса об индустрии видеоигр начала XXI века. В статье — обзор платформ, продаж по жанрам, региональных предпочтений и отдельный акцент на жанре RPG.

Прежде чем перейти к аналитике для статьи, нужно подготовить данные: проверить корректность, провести предобработку, сделать срез по периоду 2000–2013, добавить категориальные признаки и определить топ-платформы.

## Данные

`new_games.csv` — исторические данные по продажам игр:
- название игры, платформа, год выпуска, жанр, рейтинг ESRB;
- продажи по регионам: Северная Америка, Европа, Япония, остальной мир (млн копий);
- пользовательская оценка (0–10) и оценка критиков (0–100).

## Стек

Python, Pandas, NumPy, Jupyter Notebook.

## Что сделано

**Предобработка:**
- приведение названий столбцов к snake_case;
- исправление типов данных, замена строковых значений (`unknown` и т. п.) в числовых столбцах на `NaN`, корректное приведение к числовым типам после обработки пропусков;
- расчёт абсолютных и относительных пропусков по каждому столбцу с интерпретацией их природы;
- обработка пропусков: заглушки для категориальных полей, заполнение продаж средними по платформе и году выпуска, удаление строк, не несущих информации;
- проверка явных и неявных дубликатов: нормализация регистра в названиях и жанрах, ESRB-рейтинг приведён к верхнему регистру, удалены явные дубли;
- подсчёт удалённых строк в абсолютных и относительных значениях.

**Фильтрация:**
- срез данных по периоду **2000–2013 годов включительно** в отдельный датафрейм `df_actual`.

**Категоризация:**
- `user_score_category`: высокая (8–10), средняя (3–8, без правой границы), низкая (0–3, без правой границы);
- `critic_score_category`: высокая (80–100), средняя (30–80, без правой границы), низкая (0–30, без правой границы), отдельная категория «без рейтинга» для игр с пропусками;
- топ-7 платформ по количеству выпущенных игр в актуальном периоде.

## Ключевые выводы

В результате предобработки получен очищенный датасет за 2000–2013 годы, готовый к дальнейшему исследовательскому анализу для статьи. Добавлены два новых категориальных признака — `user_score_category` и `critic_score_category` — которые упрощают сегментный анализ качества игр.

Топ-7 платформ периода определён и может использоваться как фильтр при последующем анализе продаж и предпочтений по регионам.

Структура датасета и природа пропусков дают важный сигнал для дальнейшей работы: значительная доля пропусков в оценках критиков и пользователей (особенно в ранних годах) означает, что выводы по динамике качества игр стоит делать аккуратно — данные неравномерно покрывают весь период.
